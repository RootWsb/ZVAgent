# Legal Paper Download Sources

Preferred legal sources for PDFs and full text:

- arXiv: open preprints, direct PDF via `https://arxiv.org/pdf/{id}.pdf`.
- PubMed Central: open biomedical full text when a PMCID is available.
- Unpaywall: checks whether a DOI has a legal open-access location.
- Publisher-hosted OA pages: valid when the page or metadata marks the article as open access.
- Institutional repositories: accepted manuscripts or published versions when legally deposited.
- CORE and OpenAlex: useful for finding repository links, then verify and download.

Avoid:

- Sci-Hub and mirror sites, including `https://www.sci-hub.vg/`.
- Login-only publisher PDFs unless the user already has local access and provides the file.
- Browser automation to bypass access controls.

When in doubt, download only metadata/abstract and tell the user what legal access path remains.
