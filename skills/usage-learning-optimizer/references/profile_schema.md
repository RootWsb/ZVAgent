# Usage-Learning Profile Schema

Use these files under `knowledge/profile/`.

## user_preferences.md

Stable preferences only. Add short bullets; avoid logging one-off requests.

Suggested sections:

- Global
- Research
- Crypto
- Economics
- AI Tech
- Explicit Preferences

## task_patterns.md

Reusable answer templates. Keep each template under 10 bullets.

## context_budget.md

Rules for mode selection, retrieval depth, and archive thresholds.

## feedback_log.md

Append-only raw feedback. Compact into the other files only when feedback is stable.
