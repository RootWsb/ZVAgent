# Knowledge Summary Templates

## Domain Summary

```markdown
# <Domain> Summary

> Generated/updated: YYYY-MM-DD
> Scope: <folders or pages>

## Current Map

- Theme: one-line synthesis. Sources: [A](../domain/a.md), [B](../domain/b.md)

## Stable Conclusions

- ...

## Open Questions

- ...

## Pages To Revisit

- [Page](../domain/page.md) - why it matters
```

## Page Compression

When compressing a long page, preserve:

- source URL or local source
- publication/event date
- core conclusion
- evidence and uncertainty
- links to related notes

Do not replace a source note with a summary unless the user explicitly wants deletion or archival cleanup.
