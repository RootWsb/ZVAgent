## 🔥 头条：Claude Opus 4.8 发布 + Anthropic 估值逼近万亿

### 模型升级要点

Anthropic 于美东时间周四（5 月 28 日）发布旗舰模型 **Claude Opus 4.8**，距离 Opus 4.7 仅 43 天。核心改进：

- **编码能力大幅提升**：代码缺陷未经说明就通过的概率降至前代的约 **1/4**，模型更愿意在发现风险时主动提醒用户
- **诚实性飞跃**：模型更愿意说明工作中的不确定性，减少缺乏支撑的判断——这是 Anthropic "安全可控"叙事的关键延伸
- **智能体（Agent）任务更可靠**：早期测试者反馈，Opus 4.8 执行 Agent 任务时判断更清晰，可靠性更高
- **思考强度可调节**（Effort Control）：用户可在 low / default(high) / extra / max 之间选择推理投入
- **Dynamic Workflows（研究预览）**：Claude Code 现在可以动态编写编排脚本，在单次会话中运行**数十到数百个并行 sub-agents**，面向大规模代码库迁移、安全审计等场景。典型案例——Bun 从 Zig 移植到 Rust，生成约 75 万行 Rust 代码，测试通过率 99.8%
- **Messages API 更新**：支持在 messages array 内插入 system entries，Agent 运行时可在不破坏 prompt cache 的情况下更新指令
- **价格不变**：$5/百万输入 token，$25/百万输出 token

### 融资与估值

- **H 轮 650 亿美元**，Altimeter Capital、Dragoneer、Greenoaks、Sequoia Capital 领投
- **投后估值 9650 亿美元**，逼近万亿美元关口

### 争议：蒸馏疑云

Opus 4.8 发布后即被网友发现——追问模型身份时，有时会回答自己是 **Qwen** 或 **DeepSeek**，疑似训练数据中包含其他模型的输出。这在 API 直接调用时偶现，官方客户端因有更多系统提示词约束而较少复现。Anthropic 尚未正式回应。

### 下一步

Anthropic 预告将在**数周内**推出比 Opus 更强的 **Claude Mythos 级别模型**（Mythos Preview 此前已在对齐测试中表现最佳）。

## 📄 论文与研究

### 1. RoPE 位置编码先天性缺陷（重磅）

伊利诺伊大学 + 波恩大学 + 阿贡国家实验室 + 亚马逊 AGI 团队联合论文，首次从**数学层面证明**：

- 随着文本长度增加，AI 把更高注意力分配给远处词的概率逼近 **50%**（等于随机掷硬币）
- 在 8000 词上下文中，仅 BF16 精度就存在 **超过 77,000 对位置混叠**
- 解释了为何号称支持 128K 上下文的模型在实践中仍出现位置混淆
- **影响**：需要长文档处理、复杂对话的 AI 应用需要重新审视位置编码方案

### 2. EQA-Decision（CVPR 2026）—— 北大穆亚东团队

- 400 万+ 问答对，覆盖四大推理模块
- RoboDecision 三阶段训练（SFT → CoT-SFT → GRPO）
- 8B 模型得分 68.06，**超越 GPT-5 (51.03)**
- 使具身模型实现真正的"先想后做、看图决策"

### 3. HiF-VLA（CVPR 2026）—— 西湖大学 & 浙大 & 西湖机器人

- 以 Motion 向量替代冗余像素作为动态先验
- 峰值显存仅 **31.4GB**（传统方法 63.6GB）
- 推理延迟保持恒定极低
- CALVIN / LIBERO-LONG 长程任务成功率显著超越 SOTA

### 4. AGI 新定义："人工科学家"框架

- 澳大利亚国立大学 Bennett 提出：AGI = 能在计算/记忆/能量约束下像科学家一样发现新知
- 跳出"模仿人类"范式，聚焦"主动实验 + 因果理解 + 探索与利用平衡"
- 论文虽发表于去年，但近期因 AGI 讨论升温而重新引发热议
